Illustration System

Avatars

Avatars is a doodle illustration system that I created in Figma to play with variants. In this system you can mix and match illustrations to create different avatars. Combining a different hairstyle, clothes and face expression to create a personalized avatar.

How I create the illustration system

The goal of this project was to have fun exploring figma components and doodling.

I started doodling some faces with different hairstyles, clothes and face expressions separately on a piece of paper. I used a head template to create all the different body parts with the same size. You can check some examples in the images below.

Then, once I was happy with all the different variations, I took a photo of all the doodles and vectorized them in Illustrator, creating individual body parts.

These are all the final expressions, hairstyles and clothes vectorized:

As a last stept, I created different variants in Figma so we could combine all the different parts of the body as we like, to create our own avatar.
